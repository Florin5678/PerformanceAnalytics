# Student Performance Dashboard

A data visualization project analyzing student performance in three different subjects based on physiological and socioeconomic factors.

## Tools Used
- MySQL for data storage and querying
- Tableau for data visualization
- Dataset 1: [https://www.kaggle.com/datasets/rkiattisak/student-performance-in-mathematics](https://www.kaggle.com/datasets/rkiattisak/student-performance-in-mathematics)
- Dataset 2: [https://www.kaggle.com/datasets/lainguyn123/student-performance-factors](https://www.kaggle.com/datasets/lainguyn123/student-performance-factors)
## How to Use
1. Import the SQL file into MySQL.
2. Open the Tableau workbook (`.twb`) and connect to the MySQL database.
3. Interact with filters to explore insights on how various factors affect student performance and promovability
